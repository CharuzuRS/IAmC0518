What is TGP?

TGP is a Minecraft Server based in Singapore, it holds 4 large servers, SkyWars, BedWars, Survival-Games, RPG-Survival. This server has the capability to entertain young players with smart mind-games, and whats best is that the gameplay itself is based from scratch creation by the entire network development team.
